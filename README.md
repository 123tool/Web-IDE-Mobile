# 📱 CodeDEV - Mobile Web IDE

**CodeDEV** adalah editor kode HTML/CSS/JS sederhana yang dirancang khusus untuk perangkat seluler. Dibuat untuk memudahkan developer melakukan *live testing* kode langsung dari genggaman.

## 🚀 Live Demo
Lihat hasilnya di sini: `https://username-kamu.github.io/nama-repo/`

## ✨ Fitur Utama
- **Dual Mode:** Beralih antara Editor dan Pratinjau dengan satu ketukan.
- **Instant Render:** Hasil kode langsung muncul tanpa perlu reload.
- **Mobile First Design:** UI minimalis yang dioptimalkan untuk navigasi layar sentuh.
- **Clean Interface:** Menggunakan estetika modern dengan dukungan font monospaced (Fira Code).

## 🛠️ Cara Menjalankan di GitHub Pages
1. Buat repositori baru di GitHub.
2. Upload file `index.html` ke repositori tersebut.
3. Masuk ke tab **Settings** > **Pages**.
4. Di bagian **Build and deployment**, pilih branch `main` dan klik **Save**.
5. Tunggu beberapa saat, dan link demo kamu akan aktif!
